

# PhotoStudio — Premium Photography Portfolio & Booking System (Phase 1: Core)

## Design Foundation
- **Dark cinematic theme**: Black/dark gray base with neon purple and gold gradient accents
- **Typography**: Bold luxury headings with minimal spacing
- **Effects**: Glassmorphism cards, cinematic gradients, depth shadows, smooth animations
- **Layout**: Geometric sections with diagonal dividers, overlapping cards, grid-based symmetry

---

## 1. Hero Section
- Full-width cinematic background image with soft dark overlay
- Floating geometric shapes with subtle CSS animations
- Animated headline with fade-in effect
- Two CTA buttons: "Book a Shoot" and "Explore Portfolio"
- Parallax-style scrolling depth

## 2. Portfolio / Gallery
- Masonry gallery layout with category filter tabs (Wedding, Pre-Wedding, Fashion, Product, Event, Drone, Cinematic Videos)
- Hover effects: image zoom, photographer info overlay, light flash animation
- Before/After comparison slider component for photo editing showcase
- All using mock/placeholder images

## 3. Services Section (including Drone)
- Service cards with glassmorphism styling for all photography categories
- Dedicated Drone Photography & Videography section with:
  - Aerial shoot package cards
  - Drone specs cards (flight duration, camera resolution, coverage area, pilot experience)
  - Equipment showcase with hover details
  - Preview gallery grid

## 4. Equipment Rental Module (Frontend Mock)
- Equipment cards for DSLR, Lenses, Lighting, Drones, Tripods, Studio Setup
- Multi-step rental flow:
  - Select equipment → Choose structure type (Indoor/Outdoor/Event/Wedding)
  - Pick time duration (Hourly/Half Day/Full Day/Multi-day)
  - Calendar date picker
- Dynamic price calculation preview (using mock pricing data)
- Booking summary card

## 5. Previous Projects / Our Work
- Timeline-style project showcase with large cinematic cards
- Each card shows: client name, project type, location, gallery preview thumbnails
- Geometric framing and hover animations

## 6. Additional Sections
- **Photographer Team**: Profile cards with specialization badges
- **Awards & Achievements**: Counter animations with icons
- **Client Testimonials**: Styled video-style testimonial cards
- **Brand Collaborations**: Auto-scrolling logo slider

## 7. Navigation & Layout
- Sticky header with transparent-to-dark scroll effect
- Smooth scrolling between sections
- Mobile-responsive hamburger menu
- Footer with contact info and social links

## 8. Animations & Micro-interactions
- Page section reveal animations on scroll (CSS-based, no Framer Motion to keep dependencies light)
- Card hover interactions (scale, glow, zoom)
- Cursor glow effect on gallery images
- Animated number counters
- Loading skeleton placeholders
- Smooth scroll behavior

> **Phase 2 (later)**: Photographer & Customer dashboard UIs, video testimonials, booking confirmation animations, and additional polish.
